# bilibili_show
#bilibili会员购漫展演出
bilibili会员购中的漫展演出信息查询与储存，以及bilibili会员漫展演出中的购票等等。
当前使用的是python语言进行查询与购票。
抢票功能仅支持需要实名制购买的票，如：哔哩哔哩BML，哔哩哔哩BW，cp等等。
查询功能仅支持会员购漫展演出。

目前购票功能不完善，等具体的预售或购买页面出来的再进行最终确定。
